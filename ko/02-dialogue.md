# 💬 공론장과 토론
{: .no_toc }

시민이 함께 대화하고 의견을 나누는 다양한 방식. 빠띠는 디지털 도구와 함께, 그 도구를 가장 효과적으로 활용하는 이벤트 포맷과 운영 방법론을 10년간 발전시켜 왔습니다.

## 플랫폼 기능
{: .text-delta }

시민이 일상적으로 사용하는 토론 도구들.

### 1. 채널별 토론

✅ 구현 완료
{: .label .label-green }

주제·이슈별 채널을 구성해 구성원이 의견을 나누는 공간. 관심 주제별로 자율적인 토론이 가능합니다.

[캠페인즈에서 보기](https://campaigns.do){: .btn .btn-outline .fs-3 }

### 2. 제안 게시판

✅ 구현 완료
{: .label .label-green }

누구나 제안을 올리고 공감·댓글로 의견을 나누는 기본 공론 공간.

[제안 게시판 사례 (e프포레스트)](https://epforest.kr/1/proposal/list){: .btn .btn-outline .fs-3 }

### 3. 뉴스 코멘트

✅ 구현 완료
{: .label .label-green }

뉴스 기사에 직접 코멘트를 달고 토론으로 연결하는 기능. 시민이 뉴스를 단순히 소비하는 것을 넘어 능동적으로 해석하고 토론할 수 있게 합니다.

[캠페인즈 뉴스](https://campaigns.do/articles){: .btn .btn-outline .fs-3 }

### 4. 공동 위키 문서

✅ 구현 완료
{: .label .label-green }

구성원 누구나 편집 가능한 공동 작업 문서. 시민이 함께 지식과 자료를 축적합니다.

### 5. 실시간 의견 시각화

✅ 구현 완료
{: .label .label-green }

투표·토론 현황이 화면에 실시간으로 표시되어 토론 흐름을 모두가 함께 봅니다. 의견의 변화가 즉시 시각화됩니다.

[빠띠 타운홀](https://townhall.kr){: .btn .btn-outline .fs-3 }

### 6. 응원하기 (실시간 리액션)

✅ 구현 완료
{: .label .label-green }

발표자에게 실시간 박수·응원을 보내 행사를 극적으로 연출하는 기능. 디지털 공론장에 시민의 온기를 더합니다.

[빠띠 타운홀 소개](https://townhall.kr/introduction){: .btn .btn-outline .fs-3 }

### 7. 실시간 제안·질문 수집

✅ 구현 완료
{: .label .label-green }

행사 중 시민의 제안과 질문을 즉시 수집하는 기능. 토론·강연·행사를 양방향으로 만듭니다.

[빠띠 타운홀 소개](https://townhall.kr/introduction){: .btn .btn-outline .fs-3 }

## 이벤트 포맷·방법론
{: .text-delta }

빠띠가 10년간 개발하고 운영해온 공론장과 대화 프로그램.

### 8. 별별대화

✅ 구현 완료
{: .label .label-green }
⭐ 빠띠 오리지널
{: .label .label-purple }

은하투표 응답 데이터를 바탕으로 가장 다른 의견을 가진 시민을 1:1 또는 소규모로 매칭해 대화를 진행하는 빠띠 고유의 숙의 프로그램. 대화 전후 의견 변화를 추적합니다. 대표적인 활용 사례로 한겨레와 함께 운영한 한국의 대화가 있습니다.

[한국의 대화](https://talks.campaigns.do/goals/parti-talks){: .btn .btn-outline .fs-3 }

### 9. 정책배틀

✅ 구현 완료
{: .label .label-green }

두 정책 의견을 맞세워 참가자가 실시간으로 지지를 선택·변경하는 토론 포맷. 전문가 발표, 그룹 토론, 시민배심원단 투표, 대안 제안의 흐름으로 진행됩니다. 의견의 흐름을 모두가 함께 봅니다.

[정책배틀 소개 글](https://medium.com/parti-xyz-developers/%EB%B9%A0%EB%9D%A0-%ED%83%80%EC%9A%B4%ED%99%80-x-%EC%A0%95%EC%B1%85%EB%B0%B0%ED%8B%80-%EC%9E%AC%EB%AF%B8%EC%9E%88%EA%B3%A0-%EB%AF%BC%EC%A3%BC%EC%A0%81%EC%9D%B8-%EA%B3%B5%EB%A1%A0%EC%9E%A5-%EB%A7%8C%EB%93%A4%EA%B8%B0-b25ccaae9cc7){: .btn .btn-outline .fs-3 }

### 10. 아이디어마켓

✅ 구현 완료
{: .label .label-green }

아이디어를 제안·전시하고 투표로 선호를 결정하는 행사 포맷. 시민의 창의적 아이디어를 모으고 발전시키는 마당.

### 11. 브레인스토밍 툴킷

✅ 구현 완료
{: .label .label-green }

소그룹 토론 후 의견을 모아 전체 투표로 연결하는 구조화된 대화 프로그램. 흩어진 아이디어를 집단의 결정으로 만드는 방법론.

### 12. 열린 토론회·온라인 시민대화

✅ 구현 완료
{: .label .label-green }

격주로 운영되는 정기 온라인 공론 대화 프로그램. 시민이 일상적으로 공론에 참여할 수 있는 공간.

### 13. 이슈밋업

✅ 구현 완료
{: .label .label-green }

이슈 전문가 중심으로 멤버들이 모여 소규모 대화를 나누는 멤버십 프로그램. 깊이 있는 주제 탐구를 위한 모임.

## 향후 과제
{: .text-delta }

빠띠가 다음 10년 동안 발전시켜 가고자 하는 방향.

### 14. 논거 구조화 토론

⬜ 향후 과제
{: .label }

찬반 주장에 근거·출처·반론을 계층 구조로 첨부해 '이유 있는 토론'을 유도하는 모듈. 단순 의견 교환을 넘어 논리적 검증이 가능한 토론 환경.

### 15. AI 토론 요약

⬜ 향후 과제
{: .label }

장시간 토론의 핵심 주장·합의점·쟁점을 자동 요약해 참가자에게 공유. 토론의 양이 늘어나도 참여 비용이 늘지 않도록.
