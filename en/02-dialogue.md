# 💬 Public Forums & Dialogue
{: .no_toc }

Diverse ways for citizens to dialogue and exchange opinions together. Parti has developed not only digital tools but also the event formats and operational methodologies that make these tools most effective — refined over a decade of practice.

## Platform Features
{: .text-delta }

Daily-use dialogue tools available on Parti's platforms.

### 1. Topic-Based Channels

✅ Implemented
{: .label .label-green }

Channels organized by topic or issue where members exchange views. Autonomous discussion by area of interest.

[View on Campaigns.do](https://campaigns.do){: .btn .btn-outline .fs-3 }

### 2. Proposal Board

✅ Implemented
{: .label .label-green }

A basic public forum where anyone can post proposals and engage through reactions and comments.

[Example: e-Forest Citizen Platform](https://epforest.kr/1/proposal/list){: .btn .btn-outline .fs-3 }

### 3. News Comments

✅ Implemented
{: .label .label-green }

A feature for commenting directly on news articles and converting them into discussions. Moves citizens from passive news consumption to active interpretation and dialogue.

[News on Campaigns.do](https://campaigns.do/articles){: .btn .btn-outline .fs-3 }

### 4. Collaborative Wiki Documents

✅ Implemented
{: .label .label-green }

Co-authored documents that any member can edit. Citizens build shared knowledge and resources together.

### 5. Real-Time Opinion Visualization

✅ Implemented
{: .label .label-green }

Voting and discussion status displayed live on screen so everyone witnesses the flow of dialogue together. Opinion shifts become immediately visible.

[Parti Townhall](https://townhall.kr){: .btn .btn-outline .fs-3 }

### 6. Live Reactions (Cheering)

✅ Implemented
{: .label .label-green }

Real-time applause and cheers sent to speakers, creating dramatic atmosphere during events. Adds civic warmth to digital public squares.

[Parti Townhall Introduction](https://townhall.kr/introduction){: .btn .btn-outline .fs-3 }

### 7. Real-Time Suggestions & Questions

✅ Implemented
{: .label .label-green }

A feature that collects citizen suggestions and questions during live events. Transforms talks, lectures, and gatherings into two-way exchanges.

[Parti Townhall Introduction](https://townhall.kr/introduction){: .btn .btn-outline .fs-3 }

## Event Formats & Methodologies
{: .text-delta }

Public forum and dialogue programs Parti has developed and operated over the past decade.

### 8. Star Talk

✅ Implemented
{: .label .label-green }
⭐ Parti Original
{: .label .label-purple }

Parti's unique deliberative program that matches citizens with the most differing views — based on Galaxy Vote response data — for one-on-one or small-group dialogue. Tracks opinion shifts before and after dialogue. A signature implementation is Korea Talks, operated jointly with Hankyoreh.

[Korea Talks](https://talks.campaigns.do/goals/parti-talks){: .btn .btn-outline .fs-3 }

### 9. Policy Battle

✅ Implemented
{: .label .label-green }

A debate format where two opposing policy arguments face each other and participants choose and change their support in real time. The format includes expert presentations, group discussions, citizen jury voting, and alternative proposals. The flow of opinion is visible to all.

[About Policy Battle](https://medium.com/parti-xyz-developers/%EB%B9%A0%EB%9D%A0-%ED%83%80%EC%9A%B4%ED%99%80-x-%EC%A0%95%EC%B1%85%EB%B0%B0%ED%8B%80-%EC%9E%AC%EB%AF%B8%EC%9E%88%EA%B3%A0-%EB%AF%BC%EC%A3%BC%EC%A0%81%EC%9D%B8-%EA%B3%B5%EB%A1%A0%EC%9E%A5-%EB%A7%8C%EB%93%A4%EA%B8%B0-b25ccaae9cc7){: .btn .btn-outline .fs-3 }

### 10. Idea Market

✅ Implemented
{: .label .label-green }

An event format where ideas are proposed, displayed, and selected by vote. A marketplace for gathering and developing citizens' creative ideas.

### 11. Brainstorming Toolkit

✅ Implemented
{: .label .label-green }

A structured dialogue program that connects small-group discussions to a collective vote. A methodology for turning scattered ideas into group decisions.

### 12. Open Forum · Online Citizen Dialogue

✅ Implemented
{: .label .label-green }

A regular biweekly online dialogue program. A space where citizens can participate in public discourse as part of everyday life.

### 13. Issue Meetup

✅ Implemented
{: .label .label-green }

Small-group dialogue gatherings centered on issue experts, organized as a membership program. A space for deeper exploration of specific topics.

## Future Directions
{: .text-delta }

What Parti aspires to develop in the next decade.

### 14. Argument-Structured Debate

⬜ Future Direction
{: .label }

A module that attaches evidence, sources, and counter-arguments to claims in a hierarchical structure, encouraging "reasoned debate." Moves beyond opinion exchange toward a debate environment with logical verifiability.

### 15. AI Discussion Summary

⬜ Future Direction
{: .label }

Automatic summarization of key arguments, points of agreement, and contested issues from long discussions, shared with participants. Ensures participation cost does not grow with discussion volume.
